# 2D Water Simulations & Fluid Dynamics

A collection of real-time 2D physics simulations and mathematical documentation exploring fluid dynamics, ranging from simple mass-spring surface waves to advanced Navier-Stokes particle hydrodynamics and partial differential equations.

## 🌊 Included Models

1. **Mass-Spring System** (`Sivulta kuvattu jousivesi`)
   - 2D heightfield wave motion based on Hooke's Law and spring connections.
2. **Interactive Buoyancy** (`2D Veden aaltoliike`)
   - Surface waves integrated with rigid body floating mechanics (Archimedes' principle).
3. **Smoothed Particle Hydrodynamics (SPH)** (`SPH Partikkelivesi NavierStokes`)
   - Particle-based fluid simulation resolving density, pressure, and particle forces.
4. **Shallow Water Equations (SWE)** (`Matalan veden yhtälöt`)
   - Grid-based numerical solver for non-linear shallow water equations.
5. **Technical Documentation** (`Reaaliaikaiset 2D-vesisimulaatiot`)
   - Full mathematical overview with rendered MathJax LaTeX equations.
